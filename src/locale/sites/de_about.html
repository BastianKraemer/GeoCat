Kurzanleitung

Was ist GeoCat?

GeoCat ist eine Quelloffene (Open-Source) Geo-Caching Anwendung, die vollständig im Browser läuft und auf aktuelle Webtechnologien wie HTML5 setzt.

Diese Anwendung wurde im Rahmen eines Studienprojektes entwickelt. Für das Kennenlernen von GeoCat gilt insbesondere bei der Nutzung auf Smartphones: „Einfach ausprobieren“
Alle Icons wurden nach bestem Gewissen verwendet, um eine intuitive Bedienung zu ermöglichen.

Sie können GeoCat gefahrlos ausprobieren und auch unbeschriftete Icons gefahrlos nutzen. In den meisten Fällen wird deren Bedeutung anhand der Icons unmittelbar verständlich, andernfals hilft die daraus folgende Aktion weiter. Bei der Veränderung oder gar dem Löschen von Daten erfolgt in jedem Fall eine Rückfrage, welche eine detaillierte Beschreibung enthält.

Was brauche ich zum Loslegen?

Es sind nur drei Dinge erforderlich:

  1. Ein GeoCat Benutzeraccount - die Anmeldung dauert weniger als eine Minute.
  2. Ein aktueller, HTML5-fähiger Webbrowser (siehe "Unterstützte Browser") sowie GPS-Unterstützung für die Navigation
  3. Ein voller Akku: Das Verfolgen und Abfragen der GPS-Position erfordert relativ viel Energie
Während der Navigation ist keine aktive Internetverbindung erforderlich. Diese wird nur benötigt, sobald man ein Cache erreicht hat. Für die Wegstrecke zwischen den Caches kann der Internetzugang deaktiviert werden.

Geo-Caching und Geo-Tracking

Geo-Caching
bezeichnet eine Art Schnitzeljagd mithilfe von GPS, dabei ist der Weg zum Ziel nicht bekannt, nur dessen GPS-Koordinaten. Anhand der eigenen Position lässt sich also die Richtung zum Ziel bestimmen.
Geo-Tracking
Der Begriff Geo-Tracking bezeichnet das Verfolgen/Aufzeichnen der eigenen Position, um anschließend die zurückgelegte Strecke ansehen zu können

Tipps und Tricks

Die Navigation mit dem GeoCat Kompass funktioniert auf den meisten Geräten sehr gut, es gibt aber ein paar Dinge, die man bei der Verwendung beachten sollte:

Da GeoCat als reine Webanwendung im Browser läuft, ist es nicht möglich den internen Kompass abzufragen, welcher in den meisten Smartphones verbaut ist. Aus diesem Grund kann die Bewegungsrichtung nur dann bestimmt werden, indem man sich bewegt. Der GeoCat Kompass und die dargestellten Himmelsrichtungen sind also nur solange wirklich korrekt, wie man gleichmäßig in eine Richtung läuft. Eine Drehung auf der Stelle führt zu einer fehlerhaften Darstellung.

Außerdem kann es vorkommen, dass sich bei einer Zoom-Geste oder einer Bildschirmdrehung die Größe des Kompasses auf eine winzige Fläche reduziert. Dies lässt sich aber durch ein simples erneutes Drehen des Gerätes beheben.

Wo werden meine Daten gespeichert?

Die Verarbeitung Ihrer Positionsdaten ist ein zentraler Bestandteil von GeoCat, die Auswertung der GPS-Position erfolgt dabei aber immer lokal und wird nur mit Ihrer expliziten Zustimmung zum Server übertragen, dies kann bei folgenden Situationen vorkommen:

  • In einem Wettbewerb:
    Sie markieren eine Position als 'erreicht'. In diesem Fall wird gespeichert, wann Sie diesen Punkt erreicht haben.
  • Beim Einstellen von neuen Zielen:
    Wenn Sie ein neues Ziel auf dem Server speichern, haben Sie die Möglichkeit Ihren aktuellen Standort zu verwenden.
  • Durch Aktivieren der Tracking-Funktion:
    GeoCat bietet einen Tracking Modus, der Ihre Position alle 30 Sekunden an den GeoCat Server überträgt und Ihren Freunden zur Verfügung stellt, somit können Sie direkt zu Ihren Freunden navigieren oder live deren Bewegung verfolgen (sofern die Tracking-Funktion aktiv ist und Ihnen die entsprechende Berechtigung vergeben wurde)

Welche Browser werden unterstützt?

Prinzipiell kann jeder HTML-5 fähige Browser verwendet werden. Getestet wurden folgende Browser:

  • Firefox 40+ sowie Firefox Mobile
  • Microsoft Edge
  • Google Chrome
Hinweis: Alle Versionen des Internet-Explorers bieten keine ausreichende HTML5 Unterstützung für GeoCat.
