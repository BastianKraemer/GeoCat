Brief introduction

What is GeoCat?

GeoCat is an open source Geo-Caching application which is running in your web browser by using latest technologies like HTML5.

This application has been developed within the context of a study project. If you want to learn how to use GeoCat, the best way to to this is „Trail and error“
All icons have been used in all conscience to provide an optimal usability.

Don't be afraid of unlabeled buttons or icons, infer the actions in the most cases out of the context. In the case you even don't know what the button is for: GeoCat will always ask you to confirm the operation before it performs actions that can't be undone.

What do I need to get started?

You just need three things:

  1. A GeoCat user account - it needs less than one minute to sign up.
  2. An up-to-date web browser wih HTML5 support (take a look at "Supported web browsers") and a device with GPS-Support
  3. A full charged battery: The GPS tracking increases the energy consumption of your device
GeoCat won't need an internet uplink as long as you are walking to any cache of a challenge. The internet connection is only necessary when you have reached your destination.

Geo-Caching and Geo-Tracking

Geo-Caching
is a kind of scavenger hunt using GPS. You don't know the route to your destination, but you know its GPS coordinates, so you can calculate the way.
Geo-Tracking
The term Geo-Tracking describes the tracking or recording of your GPS position to trace your covered track

Tips and tricks

The navigation using the GeoCat compass works very well on most devices, but there are a few things you have to know before you are getting started:

Due to the fact that GeoCat is a pure web application it isn't possible to use the internal compass which is integrated in many Smartphones. Because of this reason the only way to detect your moving direction is when you are moving. So the GeoCat compass will only display the right orientation when you are walking straightforward in the same direction for a while. If you turn on the spot, the compass will most likely display something incorrect.

Another thing that may happen is a reduced compass size after zoom gesture ore a device rotation. You can simply correct this by rotating the device again.

Where is this data stored?

To deal with your GPS position is a core feature of GeoCat, but the processing will always be performed on your device and is only sent to the server if you explicitly allow this. This can occur in the following situations:

  • In challenges:
    If you want to set a cache as 'reached'. In this case the server will store the time you have reached this cache.
  • If you are creating new destinations:
    Every time you store a new coordinate on the server you have the possibility to use your current position.
  • By activating the tracking function:
    GeoCat has a tracking mode, which will send your GPS position every 30 seconds to the GeoCat server. This is necessary to share your current position with your friends. This feature allows you to navigate to your friends or to track their movement (at least if you authorized them to do so).

Supported Browsers

Basicly you can use any browser that supports HTML5. The following web browsers are officially supported:

  • Firefox 40+ and Firefox Mobile
  • Microsoft Edge
  • Google Chrome
Note:: The HTML5 support of all versions of the Internet-Explorer is not sufficent enough to use all features of GeoCat.
